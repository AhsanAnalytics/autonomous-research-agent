# Autonomous Research & Report Agent

An autonomous AI agent that takes a goal, plans multi-step work, retrieves
knowledge from its own vector database (RAG) and the live web, calls external
tools, and produces a cited report.

## Status
In active development, built milestone by milestone.

## Stack (planned)
- LLM: Groq / Gemini / Ollama (provider-agnostic)
- Agent framework: LangGraph + LangChain
- Vector DB: Chroma
- API / UI: FastAPI + Streamlit
